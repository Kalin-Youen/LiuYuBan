# 研究文稿目录说明

这个目录用于整理“力的落差、宇宙非均匀演化、混沌与秩序”主题的研究材料。为了保证后续写作可持续、可校验，这里将内容拆成三层：

1. `理论框架_严谨版.md`
   这是主文稿。它把“已经被标准物理支持的部分”和“仍属于哲学解释或待验证假说的部分”明确分开。
2. `公式推导补充.md`
   这是推导笔记。重点修正原始草稿里维度不一致、适用条件不明确、把经典自旋与量子自旋混为一谈等问题。
3. `公式预览.html`
   这是可直接在浏览器打开的公式预览页，内置 MathJax，适合阅读和演示。
4. `论文草稿.tex`
   这是可继续扩展为正式论文的 LaTeX 草稿模板，适合后续导出 PDF。
5. `规则形成理论体系_光速与量子.md`
   这是新增的系统性文稿，把规则形成、光速、量子力学、对象稳定性和宏观涌现放进同一框架中。
6. `思考机理_认知与物理规则.md`
   这是认知延伸文稿，专门讨论思考的物理实现、想象边界，以及为什么思考内容能超出现实但思考过程仍受载体约束。
7. `意识自我_直觉灵感与错误推理.md`
   这是更高层的认知文稿，继续讨论意识、自我、直觉、灵感与错误推理如何作为同一认知系统的不同功能面出现。
8. `思考力学_与量子力学异同.md`
   这是认知动力学文稿，专门讨论思考的状态空间运作、驱动力、阈值与吸引子机制，以及它和量子力学真正相似与不同的地方。
9. `力的落差与宇宙非均匀演化_研究底稿.md`
   这是这条主线的独立 Markdown 总入口，适合单独阅读、分享和继续扩写。
10. `力的落差与宇宙非均匀演化_章节目录与总纲.md`
   这是正式写作蓝图，包含章节序号、目录与总纲，适合直接作为长文或书稿的结构框架。
11. `这些理论研究的价值_说明文章.md`
   这是一篇独立说明文章，专门论述整套研究为什么值得继续推进，以及它的理论价值、方法价值和跨学科价值。
12. `核心理论网络图谱_三十六网.md`
   这是网络化骨架文稿，把整套体系压缩为六个母网和三十六个核心理论网，适合继续做系统深化与接口扩展。
13. `核心理论网络图谱_接口深化与三十六个二级理论网.md`
   这是接口深化文稿，把十二个重点接口进一步展开为三十六个二级理论网，适合继续做案例化、变量化和章节化写作。
14. `哲学的作用之一.md`
   这是一篇哲学章节文稿，专门讨论新的物理规则如何在条件、结构、技术与认知的交织中获得创生，以及人在何种意义上能够参与这种生成。
15. `核心理论_巩固稿.md`
   这是一篇理论收束文稿，用来把整套研究压缩为最小核心命题、形式化骨架以及“硬核-软壳-越界禁区”的清晰分层。
16. `思考试验机_理论与公式生成方法.md`
   这是一篇方法论文稿，专门讨论如何把人的思考作为受约束的内部试验机，用来生成理论候选、筛选公式雏形并压缩研究程序。
17. `物质的本质是力_跨学科全景论证.md`
   这是一篇长篇论证文稿，从科学哲学、数学、物理、化学与凝聚态多个层面论证“物质的本质是力”这一命题，并明确它在现代科学中的最严格表述边界。
18. `连续性、落差与一体性_结构存在论草案.md`
   这是一篇结构存在论文稿，专门讨论宇宙连续展开、落差使对象显现、人脑依差而感知，以及“万物本为一体”在理论上如何被严格表达。
19. `作者声明_AI协作与理论牵引说明.md`
   这是一份项目声明文稿，说明全文借用 AI 技术进行协作辅助，但由作者尤安完成观点牵引、感知校验、方向修正与最终确认。
20. `星空式直观_连续整体与差结构的类别图景.md`
   这是一篇直观化文稿，用“星空式整体”的图像把连续性、落差、旋卷、物质、规则与感知分成清楚类别，适合作为整套理论的可视化入口。

## 建议的写作原则

- 把“标准物理结论”与“哲学扩展解释”分栏书写。
- 任何新定义都先做量纲检查，再说明它是否可观测、是否坐标无关、是否能给出新预测。
- 任何关于量子自旋、宇宙初始条件、物理定律起源的陈述，都不要直接写成“已被证明”，除非能给出标准文献支持。

## 目前的核心修正

- 原式 `FD = \nabla \vec{F} + \Delta U` 量纲不一致，不能直接当作基本定义。
- `\vec{F} = -\nabla U` 只适用于保守力，不能覆盖所有力学情形。
- “只要存在力的空间落差就必然产生力矩”不成立；对点粒子不成立，对刚体也需要几何结构和外场梯度共同满足条件。
- 量子自旋不是经典旋转被缩小后的版本，它来自量子态在旋转群下的表示。

## 使用方式

1. 先阅读 `理论框架_严谨版.md`，确定论文主线。
2. 再对照 `公式推导补充.md`，查看每一步的严格条件。
3. 如需演示或截图，直接在浏览器中打开 `公式预览.html`。
4. 如需排版成论文或导出 PDF，可在本地 LaTeX 环境中继续编辑 `论文草稿.tex`。
5. 如需继续扩写“规则形成”主线，优先接着写 `规则形成理论体系_光速与量子.md`。
6. 如需继续扩写“认知与想象边界”主线，优先接着写 `思考机理_认知与物理规则.md`。
7. 如需继续扩写“意识与自我”主线，优先接着写 `意识自我_直觉灵感与错误推理.md`。
8. 如需继续扩写“思考动力学与量子比较”主线，优先接着写 `思考力学_与量子力学异同.md`。
9. 如需对外分享或作为总纲使用，优先使用 `力的落差与宇宙非均匀演化_研究底稿.md`。
10. 如需正式展开章节写作，优先参考 `力的落差与宇宙非均匀演化_章节目录与总纲.md`。
11. 如需单独说明这套研究的意义和价值，优先使用 `这些理论研究的价值_说明文章.md`。
12. 如需继续做系统贯通、扩张节点和搭建理论生态，优先参考 `核心理论网络图谱_三十六网.md`。
13. 如需把十二个重点接口继续拆成二级理论网，优先使用 `核心理论网络图谱_接口深化与三十六个二级理论网.md`。
14. 如需单独讨论“新规则的生成”和“人的参与边界”，优先使用 `哲学的作用之一.md`。
15. 如需在继续扩写前先压实整套研究主轴，优先使用 `核心理论_巩固稿.md`。
16. 如需探索“如何把思考当作研究装置来推知理论与公式”，优先使用 `思考试验机_理论与公式生成方法.md`。
17. 如需集中论述“物质为何更应被理解为相互作用结构而非惰性块体”，优先使用 `物质的本质是力_跨学科全景论证.md`。
18. 如需集中展开“连续性、落差、可感性与一体性”的本体论主线，优先使用 `连续性、落差与一体性_结构存在论草案.md`。
19. 如需对外说明本项目的 AI 协作背景、作者主导关系与理论归属，优先使用 `作者声明_AI协作与理论牵引说明.md`。
20. 如需把整套理论先变成一幅可直观看懂的类别图景，再回到严格论述，优先使用 `星空式直观_连续整体与差结构的类别图景.md`。
