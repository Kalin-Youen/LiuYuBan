# 研究文稿目录说明

这个目录已经按用途重新整理。现在根目录只保留这一份总入口，其余文件分入七个子目录，分别对应 `正文`、`指正与术语`、`随笔与图景`、`议论与说明`、`总纲与图谱`、`工具与输出` 和 `书稿`。

当前结构的目标很明确：

- `正文` 放可持续扩写、可并入主书稿的正式章节；
- `指正与术语` 放概念收紧、术语决议、公式修正和边界澄清；
- `随笔与图景` 放偏直观、偏哲思、偏可视化入口的文字；
- `议论与说明` 放价值说明、作者声明和项目外层说明；
- `总纲与图谱` 放蓝图、底稿、结构网和章节骨架；
- `工具与输出` 放预览页、排版文件和导出 PDF。
- `书稿` 放完整书稿的章节文件，整合自研究材料并按统一逻辑链重写。

## 一、目录结构

### 1. `01_正文`

这一层是项目主文区，优先放正式章节与核心论证。

1. `01_正文/理论框架_严谨版.md`
   主文稿，负责区分标准物理支持部分与哲学延伸部分。
2. `01_正文/规则形成理论体系_光速与量子.md`
   系统性文稿，把规则形成、光速、量子、对象稳定性和宏观涌现并入同一框架。
3. `01_正文/思考机理_认知与物理规则.md`
   认知支线主文，讨论思考过程的物理实现与边界。
4. `01_正文/意识自我_直觉灵感与错误推理.md`
   继续展开意识、自我、直觉、灵感与错误推理的认知支线。
5. `01_正文/思考力学_与量子力学异同.md`
   认知动力学文稿，讨论状态空间、阈值、吸引子及与量子力学的真正异同。
6. `01_正文/思考试验机_理论与公式生成方法.md`
   方法论文稿，讨论如何把人的思考作为受约束的内部试验机。
7. `01_正文/核心理论_巩固稿.md`
   当前整套研究的压缩骨架，包含硬核、软壳和越界禁区。
8. `01_正文/基础理论总章_认识论升级与教育路线.md`
   基础总章，说明这套理论为何要走向认识论升级、音视频展示、实践指南和分层教育路线。
9. `01_正文/物质的本质是力_跨学科全景论证.md`
   长篇论证文稿，从科学哲学、数学、物理、化学与凝聚态层面展开。
10. `01_正文/连续性、落差与一体性_结构存在论草案.md`
   结构存在论主文，讨论连续性、落差、对象显现与一体性。
11. `01_正文/多个宇宙、连续性的唯一性与膨胀前沿_宇宙论证明链.md`
   宇宙论证明链，讨论多宇宙、连续唯一性、膨胀与终端问题。
12. `01_正文/宇宙会回缩吗_从膨胀末端到状态切换.md`
   宇宙未来命运专题，把暗能量正式纳入讨论。
13. `01_正文/持续、边界显现与宇宙性展开_第一原理重写.md`
   第一原理重写文稿，取消“不可让渡”作为核心概念，改从“持续不是占据，占据只是持续的空间表征”出发讨论边界、张力与宇宙性展开。

### 2. `02_指正与术语`

这一层负责“把话说稳”，是项目的纠偏区与术语统一区。

1. `02_指正与术语/公式推导补充.md`
   推导修正文稿，重点处理量纲、适用条件与误写问题。
2. `02_指正与术语/连续性的落地定义_从速度落差到规则形成.md`
   解决“连续太抽象”的问题，把原始直觉翻译成更稳的规则形成语言。
3. `02_指正与术语/力的定义分层_持续、落差与显现.md`
   统一“力之本、力之显、力之用”的分层定义。
4. `02_指正与术语/术语决议_持续、维继与场流.md`
   正式区分“持续、维继、场流”的职责，避免术语继续漂移。
5. `02_指正与术语/持续不是线_从维继性到结构形态的分化.md`
   说明“持续”不是线性特征，并用流体、动力系统、化学、凝聚态和微观物理中的稳定结构共识来替代不稳妥类比。

## 三、写作与整理原则

- 新增正式章节，优先放入 `01_正文`。
- 新增术语收紧、公式修正、边界澄清，优先放入 `02_指正与术语`。
- 新增偏直观、偏图景、偏哲思的短文，优先放入 `03_随笔与图景`。
- 新增价值阐述、作者说明、外层辩护，优先放入 `04_议论与说明`。
- 新增目录蓝图、结构网、接口规划，优先放入 `05_总纲与图谱`。
- 预览页、LaTeX、PDF 和其他导出件，优先放入 `06_工具与输出`。
- 新增书稿章节或对已有书稿章节的修订，优先放入 `07_书稿`。

## 四、目前最关键的内部边界

- 不把项目内部本体论语言直接写成标准物理定理。
- 不把“解释一切”的概念继续扩张成无边界口号。
- 不把“场流”直接写成现成物理场，而只把它作为对持续着的连续整体的直观命名。
- 不把宏观旋涡、天体自转和量子自旋彼此直接还原。
