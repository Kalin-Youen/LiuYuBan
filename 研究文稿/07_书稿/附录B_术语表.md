# 附录B 术语表

本附录汇总全书使用的核心术语，给出其工作定义和适用层级。除明确标为“标准物理”“标准信息论”“标准认知科学”的词外，其余术语都应优先被理解为本书内部的方法词、桥接词、历史线索或研究假说，而不是新的自然常数、基础实体或本体宣言。

尤其要避免两类误读。第一，**差异不是孤立状态本身的属性，而是在给定参照、尺度、边界和比较协议下出现的关系**；第二，**可用差异不是所有差异的裸总和，而是相对于某个系统、目标和局部状态可被转换、储存、做功或维持结构的那部分差异**。对标为“历史线索”“哲学假说”“专题压缩”的词，当前版本只保留其生成背景或启发价值，不把它们作为主判据使用。

| 术语 | 工作定义 | 适用层级 | 对应章节 |
|------|---------|---------|---------|
| **非均匀性** | 某个被指定的物理量、场量或状态变量在空间、时间或样本集合中的非恒定分布；在标准物理语境中常表现为梯度、各向异性、密度差、温度差或压力差；在本书中主要作为“差异”进入机制层时的桥接词 | 标准物理 + 桥接解释 | 2, 3, 4 |
| **差异** | 在给定尺度、参照、边界和比较规则下，两个状态、区域、路径、表述，或一个状态内部两个部分之间可被识别的不同；差异不是孤立状态本身的属性，而是比较关系；它不要求一定表现为时间中的变化，静态关系结构中同样可以成立；它先指出“哪里不一样”，还不自动构成同一条可度量、可比较或可传导的量轴 | 方法入口 | 1, 11, 白话卷 |
| **参照 / 参照对象** | 用来判断差异、方向、变化或可用性的比较基准；没有参照，单个不可分状态不能直接给出差异；同一状态相对于不同参照可呈现不同差异 | 方法入口 + 研究定义 | 1, 11 |
| **比较协议** | 规定比较对象、变量空间、尺度、距离函数、阈值和允许误差的操作规则；它决定某个“不同”是否能被登记为可比较的状态差异 | 方法论 + 判据约束 | 1, 11, 12 |
| **尺度** | 观察、建模或干预时采用的空间、时间、能量、组织或语义分辨率；许多差异只在特定尺度上可见，也只在特定尺度上可用 | 研究定义 + 标准科学桥接 | 5, 11, 12 |
| **状态差异** | 在某个已固定的变量空间 $X$ 中，两个可比较状态 $a,b$ 之间的距离或可区分度，可写作 $\Delta_X(a,b)=d_X(x_a,x_b)$；只有当变量空间、尺度、距离函数和比较对象被明确时，它才具有可比较意义 | 研究定义 + 方法入口 | 11 |
| **差异轴** | 用来说明当前比较到底固定了哪一个变量：如 $\Delta_T$、$\Delta_v$、$\Delta_C$、$\Delta_L$；不同轴上的差异不能直接比较强弱，除非先映射到同一判定量 | 研究定义 + 判据约束 | 11 |
| **差异转换** | 在给定局部状态 $z$ 下，一条差异轴通过传导、扩散、做功、反应、摩擦、辐射或耦合转为另一条差异轴的过程，可写作 $\Delta_i \xrightarrow{M_{ij}(z)} \Delta_j$；它强调“可转化，但不可任意互转”；转换受守恒、边界、耦合机制和效率损耗限制，通常伴随可用差异降级或熵产生 | 研究定义 + 机制桥梁 | 11 |
| **可用差异** | 相对于某个系统、目标、边界、资源、协议和局部状态而言，既真实存在又可被转换、储存、做功、维持结构或生成新差异的那部分差异；它更接近自由能、可维持通量、有效驱动力或可操作信息，而不是所有差异的裸总和；同一差异在不同 $z$ 下可用性不同 | 研究定义 + 判据约束 | 11 |
| **局部状态 $z$** | 在某个判据中被预先声明、可识别并会影响边界预测的局部条件切片；它可包含对象、尺度、边界、资源、协议与历史，但不是可事后无限扩展的万能容器 | 核心研究定义 + 判据约束 | 11, 12 |
| **$z$ 准入规则** | 候选变量进入局部状态 $z$ 时应至少满足可观测、可反算或有稳定代理之一，并且必须改善边界预测，而不只是降低拟合残差；否则只能停留在残差登记或工作假说层 | 方法论纪律 + 可证伪性 | 11, 12 |
| **封存态** | 结构把输入的可用差异转化、浓缩并暂时保存在局部边界、能垒或协议内的状态；它不是无损冻结，而是有损封存和延迟释放；当边界失效、能垒被跨越或释放通道打开时，封存差异会以可控释放、泄漏或失稳释放的形式出现 | 研究定义 + 机制桥梁 | 11, 专题 |
| **延迟耗散** | 结构不取消耗散，而是改写耗散的时间、地点和路径，使差异先以库存、势或梯度库的形式存在，再经可控释放、泄漏或失稳释放出现 | 桥接解释 + 机制约束 | 11, 专题 |
| **可控释放** | 封存差异在结构仍维持时，按通道、节律或需求被调度释放 | 机制桥梁 | 11, 专题 |
| **泄漏释放** | 封存差异在结构未完全失效时，通过耗散、遗忘、自放电或渗漏等方式缓慢流失 | 机制桥梁 | 11, 专题 |
| **失稳释放** | 结构边界或能垒失效后，封存差异以爆发、崩塌、短路、溃坝或大规模返工等形式集中释放 | 机制桥梁 | 11, 专题 |
| **亚稳态** | 尚未到最终平衡，但因边界、能垒、协议或组织约束而能在一段时间内维持的储能态；许多封存态都可在这个意义上理解 | 标准物理 + 桥接解释 | 5, 11, 专题 |
| **差结构** | 多个差异在空间、时间、层级或关系中的排布方式；它强调“怎么排布”，还不自动等于强弱、流向或同轴可比性 | 方法入口 + 桥接解释 | 1, 5, 11 |
| **力的落差** | 早期研究语言中对非均匀性、各向异性与梯度差异的压缩命名；当前公开入口更优先使用“差异 / 差结构 / 落差” | 历史研究语言 | 2 |
| **势能落差** | $D_U = \|U(\mathbf{x}+\ell)-U(\mathbf{x})\|/\|\ell\|$，单位长度上的势能变化率 | 标准物理 | 2 |
| **力场非均匀性** | $D_F = \|\nabla\otimes\mathbf{F}\|$，力场梯度张量的范数 | 标准物理 | 2 |
| **非均匀性描述簇** | $\mathcal{N} = (\|\nabla\rho\|, \|\nabla p\|, \|\nabla\otimes\mathbf{F}\|, \|\nabla T\|, A)$ | 研究方法 | 2 |
| **力之本** | 早期哲学线索中的压缩词，用来表达持续与张力背景；保留作历史线索，不作为当前公开入口、物理解释或方法判据的主干术语 | 早期哲学线索 | 2 |
| **力之显** | 早期哲学线索中对方向性、张力性与可作用性的压缩表达；当前版本仅在追溯旧线索时使用，不作为当前主干术语 | 早期哲学线索 | 2 |
| **力之用** | 早期表述中试图回到相互作用、场、约束和传播机制的桥接词；当前版本在标准物理语境中应直接使用“相互作用、场、约束、传播机制”等术语，而不以“力之用”替代 | 历史桥接语言 | 2 |
| **持续** | 对延续、维持、传递与后续展开条件的压缩表达；当前公开版本更适合作为桥接词，而不作为先验终极本体词 | 桥接解释 + 历史线索 | 2, 4 |
| **维继** | 对持续性的动态化表达，强调延续、接续与保持；主要用于追溯早期表述，不充当当前主线的首要总纲 | 哲学过渡 + 历史线索 | 2 |
| **场流** | 早期文本中对“连续变化背景”的直观图景化命名，不等于标准物理中的场，也不作为当前研究判据使用；仅在解释旧稿语境时保留 | 直观图景 + 历史线索 | 2 |
| **落差** | 当差结构已经在某一明确差异轴上呈现出可比较的强弱、梯度、势差、浓度差或耦合不均时，对这种方向性不均的上位命名；不是所有差异都是落差；落差是桥接词，不预设所有落差都属于同一种传导机制 | 桥接解释 | 2, 5, 11 |
| **流** | 在某种落差、通道、介质或耦合机制存在时，状态量、物质、能量、动量、信息或影响的定向传递过程；它描述过程，不单独充当当前版本的核心词 | 动态描述 + 桥接解释 | 5, 11 |
| **结构** | 在给定尺度、边界、通量、反馈和耗散条件下，能够在一段时间内保持可识别关系的组织方式；结构不是所有变化本身，而是变化中被维持、重复或可调用的关系模式 | 研究定义 + 桥接解释 | 3, 5, 11 |
| **反馈** | 系统当前状态经由回路影响后续状态的机制；可表现为放大、抑制、延迟或纠偏，是结构维持、失稳与重组的关键中间变量 | 研究定义 + 复杂系统 | 5, 11, 12 |
| **稳态** | 某些关键变量在给定边界、通量与扰动范围内保持统计稳定的状态；它不是绝对静止，而是可维持的动态平衡或非平衡稳定；判断稳态必须说明变量、尺度、时间窗口和扰动范围 | 研究定义 + 标准物理 | 5, 11, 12 |
| **稳态窗口** | 在给定局部状态 $z$ 下，某个周期、输入、扰动或结构量能够维持系统不失稳的允许区间；窗口外不必然立刻崩塌，但系统会进入高风险、改组或失稳区 | 核心研究定义 | 11, 12 |
| **闭合方式** | 在给定尺度上，哪些变量、边界条件与反馈关系已经足以构成有效描述，而不必继续追踪更底层全部自由度；这里的闭合是相对任务和尺度的有效闭合，不是对世界的终极封闭 | 研究定义 | 5, 11, 12 |
| **时间尺度** | 某一过程、结构维持、失稳或重组显著发生变化所对应的特征时间；不同层的形成与断裂常取决于时间尺度是否分离 | 研究定义 + 标准物理 | 5, 6, 11, 12 |
| **差异张力** | 在专题讨论中对“落差”背后可转化不均的一种进一步压缩命名；它不是现成的新物理量，也不作为当前公开入口的首选总纲 | 专题压缩 + 桥接解释 | 11 |
| **组织层** | 不同尺度上因边界、耦合、粗粒化与稳定机制不同而形成的有效层次；强调尺度与机制差异，不含价值等级意味 | 研究定义 | 5, 11 |
| **失稳** | 结构或关系失去原有维持条件，开始偏离既有稳态的过程 | 复杂系统 + 桥接解释 | 5, 11 |
| **崩塌** | 某种组织方式无法继续维持时发生的快速解组、坍缩或相变；在项目中视为失稳的强形式，而非外部例外 | 复杂系统 + 哲学-理论 | 11 |
| **重组** | 差结构在失稳、解组后重新形成新的边界、通量与稳定关系的过程 | 复杂系统 + 桥接解释 | 5, 11 |
| **极限对偶性** | 在某些差结构控制轴的两端，极致集中与极端稀释可能在合适重标度下同时表现出中观组织失效、边界变量升位与低参数极限描述的弱对偶；当前只宜作为哲学—宇宙学接口假说保留，不得直接写成核心公理 | 哲学假说 + 前沿接口 | 专题, 12 |
| **秩序主线** | 在多层材料、接口与噪声中持续保留核心生成链条、主问题与优先推进方向的组织线；其任务是防止文本膨胀后重新变散 | 项目组织 + 方法论 | 1, 11, 12, AI协作卷 |
| **缺陷内生性** | 理论若想保持可成长性，就必须把自身的边界、盲区、残差、失败样本与失真风险纳入内部处理流程，而不是留在体系之外 | 方法论约束 + 理论完整性 | 1, 4, 11, 12 |
| **自作用 / 自审** | 差结构学习法也用于处理自身术语、主链、入口与协作流程；当前版本更优先使用“自审 / 自我回写”来指代这一纪律，“自作用”仅保留作早期说法；结果按吸收、补桥、降强、转专题或登记残差处理 | 方法论纪律 | 11, 12, 专题, AI协作卷 |
