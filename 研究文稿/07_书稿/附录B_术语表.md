# 附录B 术语表

本附录汇总全书使用的核心术语，给出其工作定义和适用层级。当前公开版本优先把这些术语当作学习、推理与检查的分层工具，而不是彼此可互换的本体宣言词。

| 术语 | 工作定义 | 适用层级 | 对应章节 |
|------|---------|---------|---------|
| **非均匀性** | 力场或物理量在空间上的不均匀分布 | 标准物理 + 桥接解释 | 2, 3, 4 |
| **差异** | 两个状态、区域、路径或表述之间可被识别的不同，是整套方法的第一入口；它先告诉我们“哪里不一样”，还没有自动变成同一条可比较的量轴 | 方法入口 | 1, 11, 白话卷 |
| **状态差异** | 在某个被固定的变量空间里，两个状态之间可区分的距离，可写作 $\Delta_X(a,b)=d_X(x_a,x_b)$；它是“差异”进入可比较形式后的写法 | 研究定义 + 方法入口 | 11 |
| **差异轴** | 用来说明当前比较到底固定了哪一个变量：如 $\Delta_T$、$\Delta_v$、$\Delta_C$、$\Delta_L$；不同轴上的差异不能直接比较强弱，除非先映射到同一判定量 | 研究定义 + 判据约束 | 11 |
| **差异转换** | 在给定局部状态 $z$ 下，一条差异轴通过传导、扩散、做功、反应、摩擦、辐射或耦合转为另一条差异轴的过程，可写作 $\Delta_i \xrightarrow{M_{ij}(z)} \Delta_j$；它强调“可转化，但不可任意互转” | 研究定义 + 机制桥梁 | 11 |
| **可用差异** | 在给定边界、资源与协议下，既真实存在又可用于做功、维持结构、生成新差异的那部分差异；它更接近自由能、可维持通量或有效驱动力，而不是所有差异的裸总和 | 研究定义 + 判据约束 | 11 |
| **封存态** | 结构把输入的可用差异转化并暂时保存在局部边界内的状态；它不是无损冻结，而是有损封存、延迟释放 | 研究定义 + 机制桥梁 | 11, 专题 |
| **延迟耗散** | 结构不取消耗散，而是改写耗散的时间、地点和路径，使差异先以库存、势或梯度库的形式存在，再经可控释放、泄漏或失稳释放出现 | 桥接解释 + 机制约束 | 11, 专题 |
| **亚稳态** | 尚未到最终平衡，但因边界、能垒、协议或组织约束而能在一段时间内维持的储能态；许多封存态都可在这个意义上理解 | 标准物理 + 桥接解释 | 5, 11, 专题 |
| **差结构** | 多个差异在空间、时间、层级或关系中的排布方式；它强调“怎么排布”，还不自动等于强弱、流向或同轴可比性 | 方法入口 + 桥接解释 | 1, 5, 11 |
| **力的落差** | 早期研究语言中对非均匀性、各向异性与梯度差异的压缩命名；当前公开入口更优先使用“差异 / 差结构 / 落差” | 历史研究语言 | 2 |
| **势能落差** | $D_U = \|U(\mathbf{x}+\ell)-U(\mathbf{x})\|/\|\ell\|$，单位长度上的势能变化率 | 标准物理 | 2 |
| **力场非均匀性** | $D_F = \|\nabla\otimes\mathbf{F}\|$，力场梯度张量的范数 | 标准物理 | 2 |
| **非均匀性描述簇** | $\mathcal{N} = (\|\nabla\rho\|, \|\nabla p\|, \|\nabla\otimes\mathbf{F}\|, \|\nabla T\|, A)$ | 研究方法 | 2 |
| **力之本** | 早期哲学线索中的压缩词，用来表达持续与张力背景；保留作历史线索，不作为当前公开入口的首选总纲 | 早期哲学线索 | 2 |
| **力之显** | 早期哲学线索中对方向性、张力性与可作用性的压缩表达；当前版本仅在追溯旧线索时使用 | 早期哲学线索 | 2 |
| **力之用** | 回到物理层时对相互作用、场、约束和传播机制的严格用语 | 标准物理 | 2 |
| **持续** | 对延续、维持、传递与后续展开条件的压缩表达；当前公开版本更适合作为桥接词，而不作为先验终极本体词 | 桥接解释 + 历史线索 | 2, 4 |
| **维继** | 对持续性的动态化表达，强调延续、接续与保持；主要用于追溯早期表述，不充当当前主线的首要总纲 | 哲学过渡 + 历史线索 | 2 |
| **场流** | 对持续着的连续整体的直观图景化命名（不等于标准物理场）；保留作直观辅助，不作严格定义核心 | 直观图景 + 历史线索 | 2 |
| **落差** | 当差结构已经在某一轴上呈现出可比较的强弱、梯度、势差、浓度差或耦合不均时，对这种方向性不均的上位命名；它是桥接词，不预设所有落差都属于同一种传导机制 | 桥接解释 | 2, 5, 11 |
| **流** | 落差开始定向传递时的动态结构形态；它描述过程，不单独充当当前版本的核心词 | 动态描述 + 桥接解释 | 5, 11 |
| **结构** | 在边界、反馈与耗散下暂时被维持住的组织方式；它是被留下来的东西，而不是所有变化本身 | 研究定义 + 桥接解释 | 3, 5, 11 |
| **反馈** | 系统当前状态经由回路影响后续状态的机制；可表现为放大、抑制、延迟或纠偏，是结构维持、失稳与重组的关键中间变量 | 研究定义 + 复杂系统 | 5, 11, 12 |
| **稳态** | 某些关键变量在给定边界、通量与扰动范围内保持统计稳定的状态；它不是绝对静止，而是可维持的动态平衡或非平衡稳定 | 研究定义 + 标准物理 | 5, 11, 12 |
| **闭合方式** | 在给定尺度上，哪些变量、边界条件与反馈关系已经足以构成有效描述，而不必继续追踪更底层全部自由度 | 研究定义 | 5, 11, 12 |
| **时间尺度** | 某一过程、结构维持、失稳或重组显著发生变化所对应的特征时间；不同层的形成与断裂常取决于时间尺度是否分离 | 研究定义 + 标准物理 | 5, 6, 11, 12 |
| **差异张力** | 在专题讨论中对“落差”背后可转化不均的一种进一步压缩命名；它不是现成的新物理量，也不作为当前公开入口的首选总纲 | 专题压缩 + 桥接解释 | 11 |
| **组织层** | 不同尺度上因边界、耦合、粗粒化与稳定机制不同而形成的有效层次；强调尺度与机制差异，不含价值等级意味 | 研究定义 | 5, 11 |
| **失稳** | 结构或关系失去原有维持条件，开始偏离既有稳态的过程 | 复杂系统 + 桥接解释 | 5, 11 |
| **崩塌** | 某种组织方式无法继续维持时发生的快速解组、坍缩或相变；在项目中视为失稳的强形式，而非外部例外 | 复杂系统 + 哲学-理论 | 11 |
| **重组** | 差结构在失稳、解组后重新形成新的边界、通量与稳定关系的过程 | 复杂系统 + 桥接解释 | 5, 11 |
| **极限对偶性** | 在某些差结构控制轴的两端，极致集中与极端稀释可能在合适重标度下同时表现出中观组织失效、边界变量升位与低参数极限描述的弱对偶；当前只宜作为哲学—宇宙学接口假说保留，不得直接写成核心公理 | 哲学假说 + 前沿接口 | 专题, 12 |
| **秩序主线** | 在多层材料、接口与噪声中持续保留核心生成链条、主问题与优先推进方向的组织线；其任务是防止文本膨胀后重新变散 | 项目组织 + 方法论 | 1, 11, 12, AI协作卷 |
| **缺陷内生性** | 理论若想保持完整性，就必须把自身的边界、盲区、残差与失真风险纳入内部处理，而非留在体系之外 | 方法论约束 + 理论完整性 | 1, 4, 11, 12 |
| **自作用 / 自审** | 差结构学习法也用于处理自身的术语、主链、入口与协作流程的做法；结果按吸收、补桥、降强、转专题或登记残差处理 | 方法论纪律 | 11, 12, 专题, AI协作卷 |
| **残差** | 当前主线尚未吸收完毕、但重复出现且可能具有后续接口价值的剩余问题、弱信号或解释缺口；它不等于废料，而是待分层处理的内部变量 | 研究判据 + 成长接口 | 1, 11, 12 |
| **自我失真风险** | 理论在扩写、压缩或统一化过程中，把弱命题写强、把桥接类比写成本体结论、把语言气势写成支撑强度的风险 | 方法论警戒 | 1, 4, 11, 12 |
| **反思辩证模块** | 把外部质疑转化为理论内部修正程序的可复用处理链；其标准环节包括最强化复述、类型判别、命中点定位、处理动作选择、回写与残差登记 | 方法论模块 + 成长机制 | 12, 专题 |

| **噪声** | 在理论构建与反馈流中暂时无序、弱连接或未归类的差异成分；不应一律删除，而应区分可耗散噪声、待观察噪声与可能转化为信号的噪声 | 方法论 + 协作系统 | 12, AI协作卷 |
