# 第7章 一套今天就能用的看事顺序

> 先找不同，再划边界，再试做法，再看反馈，再留下有用的东西。

## 7.1 这卷到这里，其实只有一条顺序

前面讲了差别、边界、重复有效、消息身体环境、想法草稿、方法外置。听起来有好几件事，其实可以收成一条很短的顺序：

1. 先找哪里不一样。
2. 再划清这次只处理什么。
3. 找一个小做法去试。
4. 看反馈，不靠感觉硬猜。
5. 有效的留下，无效的放下。

这条顺序不神秘，但很耐用。学习能用，工作能用，关系能用，问 AI 也能用。

## 7.2 用在学习上

比如你学一章数学。

不要先说“我学不会”。先找差别：这类题和例题哪里不同？再划边界：今天只处理读题，还是只处理公式选择？然后试一个小做法：先做 3 题，每题写一句“关键条件是什么”。做完看反馈：错在条件、公式，还是计算？最后留下一个下次能用的小模板。

这样学习就从“我不行”变成了“我知道下一轮改哪里”。

## 7.3 用在工作上

比如一个任务迟迟推不动。

先找差别：它和以前能完成的任务哪里不一样？是目标不清，资源不够，还是等待别人？再划边界：这周只交什么，不交什么？然后试一个小动作：先做出最小版本，发给相关人确认。看反馈：对方要的是方向修改，还是细节修改？最后把确认过的流程写进清单。

工作变顺，常常不是因为你突然更拼，而是因为你少做了很多没边界的消耗。

## 7.4 用在关系上

比如你和一个人反复沟通不顺。

先找差别：这次和以前哪里不一样？是语气、时间、话题，还是期待？再划边界：这次只谈一个具体行为，不谈对方整个人。然后试一个小动作：先复述对方意思，再说自己的边界。看反馈：对方是没听懂，还是不愿意尊重？最后决定下一次怎么调整距离。

关系里最怕的不是有问题，而是把所有问题都混成一句“你总是这样”。

## 7.5 用在 AI 上

问 AI 也一样。

如果它答偏了，不要只说“你重新答”。先找差别：它偏在事实、语气、结构，还是目标？再划边界：这次只要摘要、方案、修改，还是检查？然后给一个清楚格式。看反馈：它是否按要求输出？最后把好用提示词留下。

一个实用格式是：

> 我的目标是：……  
> 背景是：……  
> 这次只处理：……  
> 不要处理：……  
> 请用……格式输出。  
> 不确定的地方请列出来。

这比一句“帮我看看”稳定得多。

## 7.6 今天就能做的小动作

拿今天最卡的一件事，照这个表写：

| 问题 | 我的答案 |
| --- | --- |
| 哪里不一样？ |  |
| 这轮只处理什么？ |  |
| 我先试哪个小动作？ |  |
| 怎么看反馈？ |  |
| 有用的话怎么留下？ |  |

不用写漂亮，能让你开始动就行。

## 7.7 本章只留下这一句

这卷真正想给你的，不是一套大词，而是一条顺手的生活流程：

> 找差别，划边界，小步试，看反馈，留下有效做法。
