# 生活卷写作备忘录

这份备忘录写给后续继续参与本卷的 AI 与写作者。它只固定一件事：生活卷要让人马上看懂、马上能转述、马上能拿去试。

## 一、性质

生活卷不是讲解课，也不是旧内容的简短版。

它更像一组经过筛选的生活经验：读书怎么抓入口，做事怎么分轻重，关系怎么划边儿，反馈怎么留下，AI 怎么帮学习而不替你思考。

一句话：生活卷要先给人几步能用的日常动作。

## 二、写作原则

- 先给生活场景，再给判断动作。
- 先让人看懂，再让人愿意复述。
- 先留下能做的事，再决定要不要解释更多。
- 少用大词，少做宣布，少让读者觉得自己在上课。
- 不安排对照页码，不把读者赶去别处补课。
- 每段都尽量回答一句话：这能帮读者今天少卡在哪里？

## 三、内容取舍

生活卷要按优先级写，不平均铺开。

优先保留：

1. 能直接指导学习、工作、关系、自我调整的内容。
2. 能用一句话转述给别人听的内容。
3. 能立刻变成一个小动作的内容。
4. 能帮助读者删掉混乱、减少内耗、看清下一步的内容。

优先删掉：

1. 只是在展示作者想得很全的段落。
2. 只是在解释一些说法彼此怎么摆放的段落。
3. 读者离开本页就不知道怎么用的段落。
4. 重复强调“这很重要”的段落。
5. 看似深刻但不能指导行动的空泛表达。

## 四、语言要求

- 句子要短，意思要直，读起来要轻。
- 标题要像生活问题，不像论文小节。
- 多写“你可以怎么做”，少写“这说明了什么终极道理”。
- 比喻只能帮人看清，不能代替动作。
- 可以有美感，但不能牺牲清楚。
- 可以有顺滑感，但不能靠情绪推着读者跑。

最理想的句子，是读者合上页面后能这样转述：

> 哦，它其实是在说：先看哪里不一样，再划清这件事的范围，然后做一个小动作，看有没有反馈，有用就留下。

## 五、章节标准

每章最好都有四样东西：

1. 一句能带走的话。
2. 两三个生活例子。
3. 一个马上能做的小动作。
4. 一个能帮助复述的收束句。

不要求每章都解释完整。生活卷宁愿少而有用，也不要满而难转述。

## 六、禁止滑回旧写法

后续改写时，尤其要避开这些倾向：

- 把生活卷写成路线图，让读者一路去找别的页。
- 把每章都写成概念摆放。
- 用一串大词吓退读者。
- 为了显得完整，把宇宙、生命、意识、文明都强行串成大图。
- 把读者当成要被训练成研究者的人，而不是一个今天就想少绕一点的人。

生活卷可以慢慢改变人的看法，但它靠的是反复提供好用的小经验。经验汇在一起，人的看法自然会慢慢变。

## 七、最终检查

每次修改后，问五句：

1. 一个中学生能不能大概看懂？
2. 读者能不能用一句话转述？
3. 这一段有没有减少理解阻力？
4. 这一段有没有给出生活中的用法？
5. 删掉它，会不会反而更清楚？

如果第五句答案是“会”，就删。
