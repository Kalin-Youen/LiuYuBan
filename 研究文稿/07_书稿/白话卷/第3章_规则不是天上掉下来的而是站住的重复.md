# 第3章 规则不是天上掉下来的而是站住的重复

## 3.1 人是怎么得到“规则感”的

人类有个非常厉害但又很朴素的能力：  
我们会把反复出现的关系，记成一句能指导行动的话。

比如：

- 火太近会烫；
- 东西松手会掉；
- 某种做法经常有用；
- 某个人在某种场景下大概率会那样反应。

这就是规则感。

白话说，规则并不是先写在天空上的条例，然后下来命令世界。  
人真正接触到规则的方式，通常是：

> 某种关系反复出现，而且相当稳定，于是我们把它记住。

所以从白话角度看，规则首先不是“法律条文”式的东西，  
而是“反复站得住的关系”。

## 3.2 为什么不是所有重复都能变成规则

并不是每一次重复都能变成规则。  
有些只是偶然碰巧：

- 今天这样，明天未必这样；
- 这次有效，下次不一定；
- 这个地方成立，换个地方就不成立。

真正能变成规则的，通常要满足几件事：

- 不是只偶然发生一次；
- 换几次情境，大体还能成立；
- 遇到一些小扰动，也不会立刻彻底失效；
- 能被压缩成一句比较稳定的话。

所以你主书稿里后来把规则讲成“可重复、可压缩、可稳定、可分层”，其实白话理解非常简单：

> 能反复出现、能说得清、不会轻易散架的关系，才像规则。

## 3.3 为什么规则和结构是连着的

规则不是凭空悬在结构之上的。  
很多时候，结构就是规则的承载方式。

比如：

- 一块晶体为什么有固定形态？因为内部关系稳定；
- 一个河道为什么能长期维持？因为流动和地形之间形成了稳定关系；
- 一种习惯为什么会越来越像“这个人就是这样”？因为神经和行为回路稳定下来了；
- 一套制度为什么会被执行？因为组织结构、奖惩关系和记忆方式一起把它托住了。

所以你可以把结构看成“规则站住之后的样子”，  
也可以把规则看成“结构反复维持出来的关系”。

它们不是断开的，而是前后嵌在一起的。

## 3.4 为什么耗散、反馈和边界这么重要

很多人会本能觉得：要稳定，就得一点都不耗散。  
但世界里很多真正稳定的东西，恰恰是靠持续交换和持续耗散维持的。

你可以用最简单的生活感觉去懂这件事：

- 火焰一直在散失能量，但它的形态能持续一段时间；
- 人体一直在耗能，但生命正是靠这种耗能维持；
- 城市天天在消耗资源，却正是在这种流动里维持秩序；
- 一个团队一直有信息流、情绪流、工作流，但好的组织会在流动里保持稳定。

所以“稳定”并不总意味着“不动”，  
更常见的情况是：

> 有些系统是在持续流动里保持样子。

这时边界、反馈和耗散就都变得重要：

- 边界让它不至于一下散掉；
- 反馈让某种关系被不断加强；
- 耗散帮它淘汰掉很多不稳定的尝试。

于是，一个本来只是暂时出现的关系，就可能慢慢被留住，最后变成规则。

## 3.5 为什么人脑也在做同样的事

规则不只存在于物理世界，也存在于我们的脑子里。

你学一件事，刚开始很乱。  
做多了以后，脑子开始抓到：

- 这类情形通常怎么处理；
- 那种错误通常怎么避免；
- 哪种变化最值得先注意；
- 什么时候该停，什么时候该转。

这就是脑子在把重复出现的稳定关系压缩成规则。

所以你主线里从物理走到认知，并不是硬拐弯。  
因为人脑本身也在做同一种事：

> 把杂乱变化里反复站住的关系，慢慢抓出来。

## 3.6 规则为什么会一层一层长上去

一条很关键的白话理解是：规则不是只有一层。

底下有底下的规则：

- 热怎么散；
- 水怎么流；
- 物体怎么落；
- 分子怎么结合。

往上又会长出新的规则：

- 生物怎么维持自己；
- 神经怎么形成习惯；
- 人怎么推理和判断；
- 群体怎么组织秩序。

它们不是彼此无关，也不是能一下压成一句话全部解释。  
更像是：

> 底下规则托住上面规则，上面规则又形成新的稳定层。

这就是为什么你的理论最后会走到“分层规则”这一步。  
白话里可以直接理解成：世界不是只有一张规则表，而是规则一层一层往上长。

## 3.7 这一章真正想让人抓住什么

这一章最重要的白话抓手有四个：

1. 规则不是先天条文，而是站住的重复；
2. 能成为规则的，不是任何重复，而是能被维持的重复；
3. 结构和规则彼此托着，不是断开的；
4. 世界里的规则不是只有一层，而是一层层长起来的。

压成一句最白的话就是：

> 规则不是天上掉下来的，它是变化被留住、关系被重复、样子被站稳之后慢慢长出来的。
