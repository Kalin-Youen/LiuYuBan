# 第3章 规则常常不是天上掉下来的，更像站住的重复

> 一句话先带着走：很多规则不是先写好的条文，更像被时间反复写稳的关系。

## 3.1 人为什么会感觉这里“有规则”

前两章讲清了不同、边界和结构，下一步自然就会碰到规则。因为人不会只满足于看见一个样子，还会继续问：下次还会这样吗？换个地方也这样吗？这里有没有稳定关系？

火太近会烫，手一松东西大体会落下，同样的方法多试几次依旧有效，某个人在某种情形下总是容易那样反应。久而久之，人会把这些反复出现的关系压成一句能指导行动的话。于是，规则感就出现了。

## 3.2 不是所有重复都配叫规则

偶然也会重复，巧合也可能连着出现几次。所以，重复本身还不够。

比较像规则的关系，通常至少有三点：它不只出现一次；它换个情境大体还能站住；它不是一堆零碎细节，而是能被压成相对清楚的读法。说得更直白一点，规则就是那些既重复、又稳定、还能被反复调用的关系。

## 3.3 规则不是压住变化，更像变化被写稳后的读法

这里最容易出现的误会，是把规则想成对变化的否定。可现实里很多规则，恰恰是在持续变化里慢慢站住的。

火焰一直在耗散，却能维持轮廓；旋涡一直在卷动，却能保住样子；生命体不停交换和消耗，却正是在这种流动里保持自己；一个组织也会在不断换人、换事、换环境时，慢慢形成比较稳定的分工和处理方式。与其把规则想成天上掉下来的条文，不如把它看成结构在时间里被反复写稳之后留下的一种读法。

也正因为如此，结构和规则常常是互相托着的。结构偏向“什么样子被留住了”，规则偏向“什么关系被反复写稳了”。看的是两个侧面，说的还是同一条线上的事。

## 3.4 这一步可以怎么拿来判断现实

如果你怀疑一个系统里是不是已经开始形成某种规则，可以先问五句：

1. 什么关系总是这样？
2. 换个情境，它还能大体站住吗？
3. 来一点扰动，它会不会立刻散掉？
4. 它能不能被压成一句可复述、可调用的话？
5. 它是只在这一层成立，还是已经被更高层继续调用？

这样你就不太容易把偶然误当规则，也不太会把一时的顺口经验误当稳定结构。

## 3.5 规则会一层层往上长

规则并不是只有一层。底下有热扩散、流动、结合和相互作用这一类规则，往上又会长出生命维持、经验压缩、推理习惯、制度协作这一类规则。底层规则托住上层规则，上层规则又在自己的尺度上形成新的稳定层。

所以这一章更想留下的，也不是一句大话，而是一个比较稳的判断：很多规则不是世界外面额外加上的说明书，而是变化被留下、关系被重复、结构被时间慢慢写稳之后，才显出来的。
