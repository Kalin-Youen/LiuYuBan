## 生活卷与研究卷的关系

生活卷不再建立章节对读映射。它只提供生活中的使用入口：先看哪里不一样，再划清这轮只处理什么，做一个小动作，看反馈，最后把有效做法留下来。

如果读者想继续进入研究卷，按问题选择即可：

| 当前问题 | 更适合进入的位置 |
|---------|----------------|
| 想压实关键概念 | `00_摘要与关键词.md`、`附录B_术语表.md`、第2章、第5章 |
| 想看物理和宇宙展开 | 第3章、第4章、第6章、第7章 |
| 想看认知和意识展开 | 第8章、第9章、第10章 |
| 想看阶段性收束 | 第11章、第12章 |
| 想把方法练进生活 | `拓展卷/` |
| 想让 AI 参与整理和回写 | `AI协作卷/` |

这张表不是生活卷章节到研究卷章节的一对一关系，只是读者带着问题继续深入时的方向提示。
