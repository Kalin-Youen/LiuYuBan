# 附录D 核心理论网络图谱

本附录给出全书核心理论的网络化骨架，采用"六个母网 + 三十六个核心理论网"的结构组织。每个理论网标注三类标签：

- **S**：标准支持网（由现有物理学、宇宙学、认知科学或复杂系统研究直接支持）
- **B**：桥接整合网（用于把不同层级理论接通，属解释性强、可继续模型化的中间网络）
- **H**：研究假说网（更适合当作研究程序、统一解释或哲学扩展，尚不能直接当作既定定律）

## 一、六个母网

| 母网 | 编号 | 核心问题 |
|------|------|---------|
| 起源与本体母网 | I | 差异从何而来？持续是什么？ |
| 结构与动力母网 | II | 差异如何被转化为结构？ |
| 规则形成母网 | III | 稳定关系如何从动力学中析出？ |
| 光速、量子与物质母网 | IV | 物理约束如何为规则提供边界？ |
| 认知与思考母网 | V | 思考如何作为高层系统出现？ |
| 意识、自我与高阶心智母网 | VI | 系统如何对自身进行全局整合和叙述？ |

母网间的依赖关系：

```
I（起源与本体）→ II（结构与动力）
I（起源与本体）→ IV（光速、量子与物质）
II（结构与动力）→ III（规则形成）
IV（光速、量子与物质）→ III（规则形成）
III（规则形成）→ V（认知与思考）
IV（光速、量子与物质）→ V（认知与思考）
V（认知与思考）→ VI（意识、自我与高阶心智）
III（规则形成）→ VI（意识、自我与高阶心智）
```
