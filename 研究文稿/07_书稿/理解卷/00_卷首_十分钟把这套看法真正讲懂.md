# 卷首 十分钟把这套看法真正讲懂

生活卷负责让你马上会用。

理解卷负责让你知道：为什么这样看，会比直接讲大道理更有用。

这卷不要求你背词，也不要求你接受一套完整理论。它只让你学会一种很普通、但很有穿透力的看法：

```text
先看哪里不一样。
再看它被什么圈住。
再看不同的东西有没有被安排起来。
再看运行时哪里会消耗。
最后看反馈回来以后，下一步改哪里。
```

这就是：

```text
差异 -> 边界 -> 结构 -> 耗散 -> 反馈
```

如果你觉得这些词还是有点硬，可以先换成生活话：

```text
哪里不一样 -> 这次只管哪一块 -> 怎么搭起来 -> 哪里最费劲 -> 结果告诉我该改哪
```

## 这套看法解决什么

它主要解决一个问题：

> 人一遇到复杂事，就容易把问题说得太大。

学不会，就说“我不适合学这个”。  
写不出来，就说“我没有表达能力”。  
关系卡住，就说“我们根本不合适”。  
工作做乱，就说“这个项目没救”。  
AI 答偏了，就说“AI 不懂我”。

这些话不一定全错，但它们太大。太大的话不能行动。

这套看法的作用，是把大话压成小动作。

不要先问“我是不是不行”。  
先问“我到底卡在哪个不同点上”。

不要先问“这个系统是不是完了”。  
先问“哪个边界漏了，哪个环节最耗，哪个反馈没回来”。

## 为什么会有放大效果

一个经验，如果只发生一次，很快会散。

你今天突然想通一件事，但没有写下来，没有试一次，没有反馈，也没有留下做法，明天它就可能变成一句模糊的感受。

但如果你把它这样处理：

1. 说清它和以前哪里不同；
2. 圈定这次只处理哪一块；
3. 把它安排进一个固定动作；
4. 看见最耗人的地方；
5. 用结果反过来改动作；

它就不再只是一次经验。

它会变成下一次可以复用的起点。你能用，别人也能听懂。别人用了以后再反馈回来，它还能继续改。

所谓倍率，不是神秘公式，而是这个过程：

> 经验被留下，做法被复用，反馈再回来，理解就开始复利。

## 这卷怎么读

不用慢慢读。

你只需要抓住四句话：

1. 漩涡说明：形状不是凭空有的，它要靠局部条件维持。
2. 水电站说明：落差本身不等于价值，要靠结构把它转出来。
3. AI 训练说明：反馈不是批评，它是下一轮修正的位置。
4. 人的学习说明：理解不是感动一下，而是能复述、能使用、能改进。

读完以后，拿一件小事试：

```text
哪里不一样？
这次只管哪一块？
我用什么把它安排起来？
哪里最费劲、最容易散？
我用什么反馈判断下一步？
```

如果能回答这五句，这卷就已经完成任务。
