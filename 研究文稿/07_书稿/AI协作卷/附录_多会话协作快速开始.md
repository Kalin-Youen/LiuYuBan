# 快速开始：多会话协作

> 让 L2/L3/L4 在多个 Trae IDE 会话中协作！

---

## 第一步：初始化系统（两个会话都执行）

在**两个 Trae IDE 会话**中都执行：

```bash
cd e:\files\code\哲学研究
python scripts/collab_tools.py init
```

你会看到：
```
✅ 协作系统已初始化: .collaboration
```

---

## 第二步：给两个会话分配角色

### 会话 A（L2 角色 · 执行型）
- 读取任务队列
- 执行 T1/T2/T3 任务
- 写入输出

### 会话 B（L3/L4 角色 · 复核/裁定型）
- 深度复核
- 最终裁定
- 决定回写位置

---

## 第三步：创建一个测试任务

在**会话 A（L2）** 中执行：

```bash
python scripts/collab_tools.py create L2 TEST \
  -f "研究文稿/07_书稿/AI协作卷/附录_L2产出_R2_P001_核心术语表v1.md" \
  -p "压缩术语表，从18个优化到15个" \
  --priority high
```

---

## 第四步：L2 查看并执行任务

在**会话 A（L2）** 中执行：

```bash
python scripts/collab_tools.py list
```

你会看到新任务，然后执行任务...

---

## 第五步：L2 提交输出

在**会话 A（L2）** 中执行任务后，写入输出：

（在 Python 中调用）
```python
from scripts.collab_tools import write_output, update_task_status, STATUS_PENDING_REVIEW

update_task_status("TASK_XXXXXXXXX", STATUS_PENDING_REVIEW)
write_output(
    task_id="TASK_XXXXXXXXX",
    output_files=["文件路径"],
    summary="任务完成摘要",
    uncertainties=["不确定点1"],
    risks=["风险点1"],
    needs_review=True
)
```

---

## 第六步：L3 复核（在会话 B）

在**会话 B（L3）** 中执行：

```bash
python scripts/collab_tools.py list -s pending_review
```

看到待复核任务后，调用：

```python
from scripts.collab_tools import write_review

write_review(
    task_id="TASK_XXXXXXXXX",
    review_comment="复核意见...",
    approved=True,
    needs_decision=True
)
```

---

## 第七步：L4 裁定（在会话 B）

最后，L4 调用：

```python
from scripts.collab_tools import write_decision

write_decision(
    task_id="TASK_XXXXXXXXX",
    decision_comment="裁定意见...",
    approved=True,
    write_back=True,
    write_back_location="研究文稿/07_书稿/AI协作卷/最终文件.md"
)
```

---

## 完整流程

```
L2创建 → L2执行 → L2提交 → L3复核 → L4裁定 → ✅完成
```

---

## 快速参考

| 操作 | 命令/函数 |
|---|---|
| 初始化系统 | `python scripts/collab_tools.py init` |
| 创建任务 | `python scripts/collab_tools.py create ...` |
| 列出任务 | `python scripts/collab_tools.py list [-s status]` |
| 更新状态 | `python scripts/collab_tools.py update <id> <status>` |
| 写入输出 | `write_output(...)` |
| 写入复核 | `write_review(...)` |
| 写入裁定 | `write_decision(...)` |

---

## 目录结构

初始化后会创建：

```
.collaboration/
├── queue/       # 任务队列
├── context/     # 共享上下文
└── output/      # 输出结果
```
