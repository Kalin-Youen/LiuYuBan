## 快捷命令（在两个会话中都可用）

| 操作 | 命令 |
|---|---|
| 列出所有任务 | `python scripts/collab_tools.py list` |
| 列出待办任务 | `python scripts/collab_tools.py list -s pending` |
| 列出待复核任务 | `python scripts/collab_tools.py list -s pending_review` |
| 更新任务状态 | `python scripts/collab_tools.py update <task_id> <status>` |

---

## Python API（推荐在 Trae IDE 中直接调用）

| 函数 | 用途 |
|---|---|
| `init_collab()` | 初始化系统 |
| `create_task(...)` | 创建新任务 |
| `list_tasks(status)` | 列出任务 |
| `update_task_status(id, status)` | 更新状态 |
| `write_output(...)` | 写入输出 |
| `write_review(...)` | 写入复核意见 |
| `write_decision(...)` | 写入最终裁定 |
