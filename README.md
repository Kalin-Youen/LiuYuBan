# 差结构学习法

## 核心思想

这个项目源于一个判断：**AI 时代所有人都会拥有更强的信息生成工具，但不是所有人都拥有更强判断力**。信息越来越便宜，判断越来越贵。

因此，这里不是在宣布"世界终极本原是什么"，也不是在再造一个"解释一切"的大词。它的实际目标更朴素：**把一套真正能帮助人学习、比较、推理、协作和修正的方法做稳**。

### 基本判断顺序

```
先看哪里不同（差异）
    ↓
再看边界、持续、反馈与耗散（差结构）
    ↓
最后判断什么能站住、能重复、能验证（规则）
```

### 六词工具链

- **差异**：把不同认出来
- **差结构**：描述不同的排布方式
- **落差**：命名可比较的强弱、梯度与不均
- **流**：落差开始定向传递时的动态形态
- **结构**：暂时被维持住的组织
- **规则**：可重复、可压缩、可验证的稳定关系

项目用"落差"把不同学科里的梯度、势差、温差、压差、浓度差与耦合不均串成同一条学习线，使它既能接入大模型训练，也能转成人类学习副本。

### 活书机制

这不是一本写完就封稿的书，而是一个**作者主导、AI 协作、读者参与**的持续构建过程：

1. **读者评论入站** → 留下进入感、卡点、修订建议与现实使用反馈
2. **AI 归并整理** → 按主线、桥梁、术语、现实接口、候写节点与版本残差归类
3. **修改模块回写** → 能落地的建议写回章节，拿不准的写入候写和残差登记卡
4. **新版本继续开放** → 章节更新后继续对外开放评论，形成循环

### 定位

差结构学习法更适合被理解成一套面向**学习者、研究者与 AI 协作者**的公共方法基础设施，而不是面向大众的大理论宣讲。

## 这是什么仓库

这个仓库不是单篇文章，也不只是一个理论展示页，而是一套正在生长的公共方法底座。当前主要分成`五卷一站`：

1. `第一卷：大众卷`
   由生活卷极速入口、理解卷和交互实验组成。它的价值，不是先把人压进一套总理论，而是在极短时间内把这套判断顺序交给读者，再用漩涡、水电站、AI 训练和生活小事把它讲成可复述、可教学、可自测的直觉动作。
2. `第二卷：研究卷`
   由摘要、主书稿、专题与术语边界组成。它的价值，是压实定义、补桥、保留验证接口，让这套方法在跨学科学习和研究里真正能站住。
3. `第三卷：拓展卷`
   由拓展卷章节、能力训练路径与方法卡组成。它的价值，是把方法从"能懂"推进到"能练、能用、能迁移"。
4. `第四卷：推演卷`
   由推演卷章节、研究卡、执行顺序与样本记录组成。它的价值，是把研究计划从展望推进到对象、变量、比较、执行与失败记录。
5. `第五卷：AI协作卷`
   由 AI 协作规则、回写协议、残差机制与训练接口组成。它的价值，是让模型协作不只会扩写，而会守主线、控噪声、记版本、促生长。
6. `一站：网站系统`
   把五卷整理成可部署、可搜索、可分页、可交互浏览的静态站点，并提供机器可读索引、知识图谱与实验入口。

它当然向大众开放，但它不以"大众哲学宣讲"作为首要目标；更现实的目标，是先把一套可复用的学习、判断、校验和协作底座做出来。

## 五卷制入口

如果你是第一次进入，先不要急着整套读完，先选卷：

1. `第一卷：大众卷`
   先读生活卷极速版，再读理解卷卷首，最后去交互实验页。目标不是先接受整套理论，而是先把这套判断顺序用起来，并且能讲给别人听。
2. `第二卷：研究卷`
   先读摘要、主书稿、专题和术语边界。目标是压实定义、主链、桥梁和验证接口。
3. `第三卷：拓展卷`
   先读 `研究文稿/07_书稿/拓展卷`。目标是把方法直接练进人的思维、学习和行动里，优先处理空想与充实、快与慢、抓关键、做优化和能动性这些问题。
4. `第四卷：推演卷`
   先读 `研究文稿/07_书稿/推演卷`。目标是把研究计划落成研究对象、目标量、样本、比较和执行顺序，优先推进差异极限、稳态周期和失稳阈值这类可继续量化的问题。
5. `第五卷：AI协作卷`
   先读 `README.md`、`研究文稿/07_书稿/AI协作卷` 和 `docs/assets/data/content.json`。目标是先守主线、边界、版本与回写纪律。

如果你只想走一条最短路径，可以按目的直接选：

1. `第一次来`
   先读生活卷极速版，用一件真实小事试一轮；如果想讲给别人听，再读理解卷卷首。
2. `想压实定义`
   先读生活卷极速版和理解卷卷首，把"找差异、划边界、搭结构、看消耗、接反馈"压成一个小流程。
3. `想看怎么实际用`
   先读生活卷极速版，再进拓展卷和 AI 协作卷第3、7、10章。
4. `想参与共建`
   先读 AI 协作卷第10章和附录模板，再去评论区留下进入感、误解点、补桥建议和现实反馈。

网站内的推荐入口：

2. 第二卷：研究卷。主书稿、理论总述、术语与边界

## 当前阅读主轴

项目当前的主要内容可以概括为：

1. 先把它当作一套判断顺序，而不是一个终极本体词。
2. 用"差异 → 差结构 → 落差 → 流 → 结构 → 规则"的分层工具链整理跨学科材料，但不把这些词彼此混写成同义总纲。
3. 优先在学习过程、人机协作、认知与可验证的跨层桥梁上落例，再把物理、生命、意识、文明与 AI 作为这套顺序的展开对象。
4. 把理论自身也纳入处理对象，对术语漂移、入口失效、桥梁过快和过强命题做自审、降强与残差登记。
5. 把理论阅读、交互检验、研究推演与版本回写逐步整合进同一个网站系统，并把它沉淀成能被学习者、研究者与 AI 协作者反复调用的公共方法基础设施。

## 五卷各自提升什么价值

把项目分成五卷，不是为了把内容拆散，而是为了让不同价值真正落地：

1. 第一卷提升的是`进入价值`和`理解价值`。它降低首次接触门槛，让人更容易上手、转述、教学和自测。
2. 第二卷提升的是`研究价值`。它把术语、桥梁、案例和验证接口压实，避免项目只剩漂亮说法。
3. 第三卷提升的是`能力价值`。它把方法从"能懂"推进到"能练、能迁移、能直接进入人的判断与行动"。
4. 第四卷提升的是`推演价值`。它把研究计划从提纲推进到对象、变量、执行、比较与失败记录。
5. 第五卷提升的是`协作价值`。它把 AI 从"扩写机器"改造成"守主线、记残差、做回写的协作系统"。

五卷合在一起，才会让这个项目同时具备进入能力、压实能力、能力构建能力、推演执行能力和持续生长能力。
