先看“落差”主轴

第一卷先负责上手

大众卷先把方法做成可进入、可复述、可自测的动作，让读者先用起来，而不是先被大词挡在门外。

第二卷负责压实

研究卷负责压定义、补桥梁、留接口，让项目从“能说”走向“能被追问、能被继续研究”。

第三卷负责协作回写

AI 协作卷负责把扩写变成受约束的协作，让版本、残差和主线能在持续修改中留下来。

三卷合起来才形成方法

先上手，再压实，再协作回写。这样项目才不是一堆说明，而是一套真的能进入现实、继续生长的结构。

Use It Now

先直接用这三张速用卡

如果你现在不想先读整卷，就先拿这三组问题去看一段文字、一个现象、一个系统，或者一条 AI 回答。它们就是第一卷最先要交给你的动作。

01 入口四问 先把不同认出来

适合刚接触一个陌生对象时使用，先别急着背定义。

  • 它和周围哪里不一样？
  • 哪种不同最稳定？
  • 哪种不同只是暂时噪声？
  • 这些不同会不会反复出现？
进入：第1章 02 结构五问 顺着边界看它怎么站住

适合分析一个系统、一段关系或一个现象是怎样被托住的。

  • 差异在哪里？
  • 边界靠什么显出来？
  • 什么在维持这条边界？
  • 哪些变化被留下，长成了结构？
  • 它会不会继续沉淀成规则？
进入：第2章 03 重复三问 看看这里有没有规则在形成

适合判断一种关系是不是已经从偶然变成了可调用的读法。

  • 什么关系总是这样？
  • 换个情境，它还能大体站住吗？
  • 它能不能被压成一句可复述的读法？
进入：第3章
